## Project Overview
This project focuses on exploring and analyzing solar farm data from **Benin**, **Sierra Leone**, and **Togo**. The objective is to evaluate solar potential across these regions and provide insights to guide strategic solar deployment.

Key goals:
- Clean and profile solar measurement datasets.
- Perform exploratory data analysis (EDA) and visualize trends.
- Compare solar potential across countries.
- Develop an interactive dashboard using Streamlit for visual exploration.
